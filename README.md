- 手順
	- Alexaのスキル名を呼びゲーム開始  
      「しりとりゲーム」
    
	- しりとりする仕方  
		- echo は最初の単語を出す
		- 単語の続きにスキル名を言う[では、次は、etc]単語を言う

	- 中断  
	 「しりとりゲーム中断」

	- 再開   
	 「しりとりゲーム再開」

	- 終了  
	 「しりとりゲーム終了」
